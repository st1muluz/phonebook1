# Телефонный справочник

Простое консольное приложение для управления контактами в телефонном справочнике.

## Описание

Это приложение предоставляет базовый функционал управления контактами: добавление новых контактов, просмотр списка существующих контактов, сохранение контактов в файл и загрузка контактов из файла. Это полезный инструмент для организации вашего списка контактов.

## Функционал

- Добавление новых контактов с указанием имени и номера телефона.
- Просмотр списка всех сохраненных контактов.
- Сохранение контактов в файл формата CSV.
- Загрузка контактов из файла при запуске приложения.
